CSS IDs

An ID is a UNIQUE identifier

ID Notes

  • In CSS an ID starts with an octothorpe (pound, hashtag, number_ - #)
  • Example uses for IDs: Target a specififc header, section, or a particular item.
  • Each ID should only be used once per page (in a whole HTML document) - they are Unique.
  • Used for one-off styling.
